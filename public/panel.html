Documentación de la API

Cómo usar la API

Endpoint de la API:

https://api.intermediahost.co/api/send
Ejemplo en PHP 
<?php
$apiKey = 'TU_API_KEY_AQUI';
$url = 'https://api.intermediahost.co/api/send';

$data = [
    'phonenumber' => '573001234567',  // Con código de país
    'text' => 'Hola! Tu factura está lista',
    'url' => 'https://tudominio.com/factura.pdf'  // Opcional
];

$ch = curl_init($url);
curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, http_build_query($data));
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    'Authorization: Bearer ' . $apiKey
]);

$response = curl_exec($ch);
$httpCode = curl_getinfo($ch, CURLINFO_HTTP_CODE);
curl_close($ch);

if ($httpCode == 200) {
    echo "Mensaje enviado correctamente";
} else {
    echo "Error al enviar mensaje";
}
?>
Ejemplo en JavaScript 
const apiKey = 'TU_API_KEY_AQUI';
const url = 'https://api.intermediahost.co/api/send';

const data = {
    phonenumber: '573001234567',
    text: 'Hola! Tu factura está lista',
    url: 'https://tudominio.com/factura.pdf'  // Opcional
};

fetch(url, {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'Authorization': `Bearer ${apiKey}`
    },
    body: JSON.stringify(data)
})
.then(response => response.json())
.then(data => {
    if (data.success) {
        console.log('Mensaje enviado:', data);
    } else {
        console.error('Error:', data.error);
    }
})
.catch(error => console.error('Error:', error));
Parámetros de la API
phonenumber Número con código de país (ej: 573001234567) Requerido
text Mensaje de texto a enviar Requerido
url URL del archivo a adjuntar (PDF, imagen, etc.) Opcional
